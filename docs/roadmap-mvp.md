# Roadmap de implementación

## Fase 0 — Fundaciones (1 semana)

- Definición de stack y arquitectura.
- Configuración de repositorio, CI, linters y entornos.
- Diseño inicial de base de datos.

## Fase 1 — MVP funcional (2–4 semanas)

- Registro/login.
- Crear encuesta con cooldown configurable.
- Votar y ver resultados inmediatos.
- Feed con scroll infinito.
- Diseño responsive mobile-first.

## Fase 2 — Calidad y retención (2 semanas)

- Tendencias.
- Encuesta del día.
- Sistema de reportes y panel básico de moderación.
- Analítica de eventos.

## Fase 3 — Monetización (1 semana)

- Integración AdSense.
- Inserción de anuncios in-feed.
- Medición de RPM/CTR y ajustes de frecuencia.

## Criterios de éxito MVP

- P95 de API de voto < 300ms.
- >= 6 encuestas respondidas por sesión media.
- Retención D1 inicial > 20%.
- Crash-free sessions > 99.5%.
